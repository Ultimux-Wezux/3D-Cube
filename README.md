🧊 Interactive 3D Cursor TrackerA custom, high-performance 3D engine built entirely from scratch using Vanilla HTML, CSS, and JavaScript. No heavy libraries—just pure math and logic.🚀 The ProjectThis project features a 3D cube that dynamically tracks the user's cursor position in real-time. It uses custom "Look-At" kinematics to calculate rotation based on the mouse's $X$ and $Y$ coordinates, providing a smooth, responsive, and immersive UI experience.Key FeaturesZero-Dependency Engine: Built from the ground up to keep the footprint tiny and the performance high.Smooth Interpolation (LERP): Implemented custom easing logic so the cube follows the cursor with a "premium" feel rather than a robotic snap.Interactive GUI: A custom settings panel that allows users to adjust:Follow Speed: Control how quickly the cube reacts to the cursor.Perspective Depth: Change the 3D depth of the field.Color & Material: Toggle styles on the fly.Responsive: Fully functional across all screen sizes.🛠️ Tech StackHTML5 (Structure)CSS3 (3D Transforms & Perspective)Vanilla JavaScript (Trigonometry-based physics & state management)💡 Why I Built ThisI’m a 13-year-old frontend developer passionate about pushing the limits of what web browsers can do. I wanted to move beyond basic websites and understand the underlying math that powers 3D graphics in the browser.This project was built to solve the challenge of creating high-quality, interactive UI elements without relying on heavy frameworks.
